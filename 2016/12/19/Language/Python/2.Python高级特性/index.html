生成器

一边循环一边计算的机制，被称为生成器：generator。

  • 方法一：把一个列表生成式的[]改成()

  • 方法二：如果一个函数定义中包含yield关键字，那么这个函数就不在是一个普通函数，而是一个generator

每次调用next()的时候执行，遇到yield语句返回，再次执行时从上次返回的yield语句处继续执行。
生成器的唯一注意事项就是：生成器只能遍历一次。
