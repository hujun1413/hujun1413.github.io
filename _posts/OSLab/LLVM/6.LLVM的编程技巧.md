## itoa用sprintf替代
```bash
```
int nValue = 80;

char* szBuffer = (char *)malloc(sizeof(int) + 1);  //分配动态内存

memset(szBuffer, 0, sizeof(int) + 1);              //内存块初始化

sprintf(szBuffer, "%d", nValue);                  //整数转化为字符串

free(szBuffer);                                    //释放动态分配的内存
```
